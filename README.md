# Threads

Threads is a simple MEAN stack implementation of the Threads app by Meta, built using Angular 2+ and NestJS.

## Technologies Used

General:

- Node.js
- TypeScript

Frontend:

- Angular 2+
- Tailwind CSS

Backend:

- NestJS
- Mongoose
- MongoDB
